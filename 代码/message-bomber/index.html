QQ消息轰炸机(附源码)
2013-08-15 / 2017-03-17
分类： 代码

学习VB的成果。

软件界面

介绍

  • 自带3个 喷人宝典 (网上搜来的),支持按行导入导出 TXT文本文档；
  • 两种轰炸模式:单文本轰炸 与 列表轰炸；
  • 可设定攻击时间间隔,不宜过短；
  • 不止是 QQ聊天 中可以用到,任何按 enter键 发送信息的工具都可以用。

使用方法

切换到聊天窗口后使用快捷键操作 ( F10 开始 | F11 停止 )
